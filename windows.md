# Comandos de terminal Windows

_Alguns comandos são iguais aos de outros SOs, mas este arquivo foca nas particularidades do Windows._

---

# Comandos do "CMD" ou Command Prompt

_Terminal mais antigo._

---

## `dir`
"diretório"  
lista o conteúdo do diretório atual  
mostra mais informações do que o comando `ls` de outros terminais

---

# Comandos do PowerShell
_Terminal mais moderno e robusto._

---

## `ls`
"list"  
lista o conteúdo do diretório atual  
tecnicamente, é um alias do PowerShell para o comando `Get-ChildItem`

---

# Comandos comuns a ambos os terminais, Command Prompt e PowerShell

---

## `cd`
"change directory"  
muda de diretório ("navega") para o diretório especificado  
quando usado sem especificar nenhum, apenas exibe o diretório atual

---

## `cls`
"clear screen"  
limpa o terminal

---

## `del`
"delete"  
usado para deletar arquivos  
deleta diretamente, sem passar por lixeira, portanto exige cuidado

---

## `mkdir`  
"make directory"  
cria um novo diretório com o nome especificado

---

## `rmdir`  
"remove directory"  
remove diretório vazio

### `/s`
"subdirectories"  
flag usada para remover um diretório e todo o seu conteúdo recursivamente

### `/q`
"questions"  
flag usada para silenciar perguntas de confirmação, que ocorreriam por padrão
